# Sales Executive Overview - Power BI Dashboard

## 📊 Project Overview
This project presents an interactive Power BI dashboard created to analyze sales and profit performance using a sales dataset. The dashboard provides clear business insights through visual analytics and helps understand performance across categories, products, regions, and years.

## 🎯 Objective
To transform raw sales data into meaningful visual insights that support data-driven decision-making.

## 🔄 Process Followed
- Imported raw dataset into Power BI
- Performed data cleaning and preparation
- Created calculated measures using DAX
- Designed interactive visualizations
- Built a business-focused dashboard layout

## 📈 Dashboard Features
- Sales by Category and Product
- Sales vs Profit Analysis
- Regional Sales Distribution
- KPI Cards for Total Sales
- Interactive Filters (Slicers)

## 🛠 Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

## 💡 Key Insights
The dashboard helps identify top-performing categories, compare yearly performance, and analyze regional contributions to overall sales.
